# Captures d’écran

Ajoutez ici uniquement des captures :

- recadrées ;
- anonymisées ;
- sans email, UUID, Project ID confidentiel, nom de client, clé, token ou mot de passe ;
- accompagnées d’un texte alternatif dans les fichiers Markdown.

Les chemins d’interface changent régulièrement : indiquez la date de la capture dans son nom ou sa légende.
